Odoo 10 : Sales & Purchase Invoice Discount

This module is used to apply discount in Sales & Purchase Order and Invoice with luxury of two Discount Methodsin Odoo.

Click here for the video tutorial

How to Use


Sales Invoice Discount

Discount in Quotations/Sale Order : User can select from 2 different Discount Method and enter Discount to be applied on whole Sale Order.



Quotations / Sale order receipt.



Discount in Invoice : User can select from 2 different Discount Method and enter Discount to be applied on whole Invoice.



Customer Invoice receipt.



Purchase Invoice Discount

Discount in Request for Quotation/Purchase Order : User can select from 2 different Discount Method and enter Discount to be applied on whole Purchase Order.



Request for Quotations / Purchase order receipt.



Discount in Invoice : User can select from 2 different Discount Method and enter Discount to be applied on whole Invoice.



Vendor Bill receipt.
